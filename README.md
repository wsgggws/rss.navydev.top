# RSS 订阅阅读平台

基于 Vue3 + TypeScript 的 RSS 订阅阅读前端项目

## 线上体验地址

- <https://rss.navydev.top/>

## 技术栈

- vue3
- ts
- vue-router
- element-plus
- axios

## .env

```.env
VITE_API_BASE_URL=http://localhost:8000
```

## Run

`npm run dev`
