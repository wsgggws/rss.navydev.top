# RSS Reader

React 18 + Vite + TypeScript RSS 阅读器，使用 animal-island-ui 组件库。

## 线上地址

- <https://rss.navydev.top/>

## 技术栈

- React 18
- Vite
- TypeScript
- animal-island-ui
- React Router v6
- Axios

## 主题

暗色赛博朋克风格，亮/暗模式切换。

## 本地开发

```bash
npm install
npm run dev
```

## 环境变量

```.env
VITE_API_BASE_URL=http://localhost:8000
```

## 构建

```bash
npm run build
```

## 后端 API

[api.rss.navydev.top](https://github.com/wsgggws/api.rss.navydev.top)
