## UI 风格

- **主题**: 暗色系 + 红色强调 (深空/赛博朋克风格)
- **主色调**: 深色背景 (#1a1a2e, #16213e) + 红色强调 (#e94560)
- **字体**: 系统字体栈，清晰易读

```

│  Header (Logo + 主题切换)                                │

│            │                                            │
```

### 组件层级

1. **App** - 根组件，主题状态管理
2. **Layout** - 整体布局框架 (Header + Sidebar + Main)
3. **Sidebar** - 订阅源列表
4. **ArticleList** - 文章列表 (杂志式)
5. **ArticleDrawer** - 文章详情抽屉 (右侧滑出)

---

## 功能模块

### 1. 订阅源管理
- 订阅源列表 (左侧边栏固定)
- 搜索过滤
- 分页

### 2. 文章列表
- 杂志式图文混排 (Medium 风格)
- 图片在左，标题 + 摘要 + 时间在右
- 点击打开文章详情

### 3. 文章详情 (Drawer)
- 从右侧滑入的抽屉面板
- Markdown 内容渲染
- 返回顶部按钮
- 外部链接

### 4. 主题切换
- 亮色 / 暗色模式
- 使用 animal-island-ui 主题系统

### 5. 404 页面
- 路由未匹配时显示

## API 接口 (保持不变)

现有 API 接口保持不变，复用后端：

- `GET /api/subscriptions` - 获取订阅列表
- `GET /api/subscriptions/:id/articles` - 获取文章列表
- `GET /api/articles/:id` - 获取文章详情
