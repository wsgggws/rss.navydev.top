animal-island-ui 组件风格

这个 UI 库的设计语言是什么？我需要先了解它的组件风格，才能提出合适的设计方案。

A

暗色系 + 红色强调 (深空/赛博朋克风格)

类似右边预览的深色背景配红色/粉色强调色，科技感强

B

亮色系 + 彩色渐变 (现代卡片风格)

类似现有 Vue 版的渐变紫蓝色调，清新明快

C

中性色 + 极简风格

黑白灰为主，简约留白，突出内容
