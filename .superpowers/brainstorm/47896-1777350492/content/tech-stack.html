React 技术栈选择

确定关键的技术决策

A

Vite + React 18 + animal-island-ui

使用 Vite 作为构建工具，React 18 新特性（Suspense, startTransition），UI 组件直接用 animal-island-ui。推荐。

B

Next.js (App Router) + animal-island-ui

使用 Next.js 框架，更适合 SSR/SEO，但 animal-island-ui 需要确认是否支持 Next.js。

C

保持 Vite + 渐进式迁移

不另起新项目，在现有 Vue 项目中通过多页或 iframe 嵌入 React 页面，逐步迁移。适合不想一次性重写的情况。
